**DUREX 0 Beads**（也称 **DUREX® 0 Beads**）是 **Orion Engineered Carbons（奥里恩工程炭黑）** 生产的一种**灯黑法（Lamp Black Process）** 特种炭黑（Specialty Carbon Black），属于**无铅（Lead-free）** 高纯度炭黑，主要用于技术橡胶制品（Mechanical Rubber Goods, MRG）。

它以**珠状（Beads）**形式供应，与粉状的 **DUREX 0 Powder** 属于同一系列，但珠状颗粒更易分散、粉尘少、加工性更好。

### 主要性能参数（典型值）

- **生产方法**：灯黑法（Lamp Black）
- **粒径**：相对较大（灯黑法特点），比表面积适中
- **结构**：中等结构
- **吸油值**：中等
- **着色力 / 黑度**：良好（提供柔和的黑度和低光泽表面）
- **分散性**：优秀（珠状形式分散均匀，挤出表面光洁度高）
- **纯度**：高纯度，无铅（Lead-free），低灰分、低杂质
- **其他**：挤出性能优异，能赋予制品**光滑表面、低光泽（low gloss）** 和均匀颜色

**关键特性**：

- **挤出性能极佳**：挤出速度快、表面光洁、尺寸稳定性好、塌陷少。
- **低光泽 + 柔和黑度**：特别适合需要美观外观的橡胶制品。
- **无铅环保**：符合RoHS、REACH等环保要求，适合对重金属有限制的场合。
- **补强效果**：属于半补强 / 着色型炭黑，补强作用中等，主要提供颜色、挤出性和加工性。

### 主要用途

DUREX 0 Beads 主要用于**技术橡胶制品（MRG）**，特别适合对**挤出表面质量和外观**有要求的场合：

- **汽车橡胶件**：密封条（weatherstrip）、窗密封条、挤出型材、软管
- **工业橡胶制品**：胶管、胶带、橡胶板、垫圈、挤出制品
- **鞋材与日用橡胶**：鞋底、橡胶配件（需要低光泽、均匀黑色的制品）
- **电缆护套与电线**：要求表面光滑的护套
- **其他**：海绵制品、减震件、要求低光泽和良好挤出性的橡胶型材

它在**挤出制品**中表现突出，能显著改善表面光洁度、降低光泽，并提供均匀的黑色外观，是许多高端密封条和型材的常用炭黑。

### 用量建议（配方参考）

**推荐用量**：**30 – 120 phr**（每百份生胶），具体取决于目标黑度、硬度和补强要求。

- **一般挤出制品 / 密封条**：**40 – 80 phr**（最常用范围）
- **高填充 / 成本控制**：**80 – 120 phr**（可与白炭黑、碳酸钙并用）
- **浅黑 / 着色为主**：**30 – 50 phr**
- **与白炭黑并用**：可改善补强与加工平衡

**典型EPDM挤出密封条配方示例**（仅供参考）：

- EPDM生胶（如Keltan 6950C或KEP系列）：100 phr
- **DUREX 0 Beads**：**50 – 80 phr**
- 白炭黑或碳酸钙：30–80 phr
- 操作油：30–60 phr
- ZnO、硬脂酸、防老剂、硫化体系：常规量
- 加工助剂（如NFS 9460 U）：1–3 phr

**优点体现**：

- 挤出表面特别光滑、低光泽、外观均匀。
- 分散性好，加工流动性佳，减少毛刺和塌陷。
- 适合浅色或黑色高端外观要求的制品。

**注意事项**：

- 储存：置于干燥处，避免潮湿。珠状形式储存稳定性好，粉尘少。
- 加工：珠状易分散，建议与油或填料一起加入。
- **食品接触注意**：DUREX 0 Beads（珠状）在某些法规下食品接触应用受限（粉状DUREX 0 Powder有更多食品接触批准），具体需查最新法规并测试。
- 实际性能以供应商最新TDS为准，**建议向Orion Engineered Carbons或当地代理索取最新技术资料**。

**与普通炭黑对比**：

- 与N550/N660等炉法炭黑相比，DUREX 0 Beads挤出表面质量和低光泽优势明显，但补强强度稍弱。
- 适合对外观要求高的挤出制品，而非高强度轮胎类。

江苏锐巴新材料科技有限公司（以下简称“江苏锐巴”）是一家专注于橡胶加工助剂研发与生产的国家级专精特新“小巨人”企业，其产品在橡胶助剂领域具有较高的市场地位。

关于你查询的 **MTT-80**，在橡胶助剂行业中，这通常指的是 **3-甲基-2-噻唑啉硫酮 (3-Methyl-2-thiazolidinethione)** 的预分散母胶（通常含量为80%）。这是一种高性能、环保型的**特种硫化剂/交联剂**。

以下是基于江苏锐巴的产品领域及行业通用标准整理的 MTT-80 性能、应用及用量指南：

### 1. 产品基本信息

- **化学名称**：3-甲基-2-噻唑啉硫酮 (3-Methyl-2-thiazolidinethione)
- **产品代号**：MTT-80（“80”通常代表有效成分含量为80%，载体通常为聚合物或蜡）
- **CAS NO**：1908-87-8
- **外观**：灰白色或淡黄色颗粒（预分散母胶形式，无粉尘）
- **性质**：属于硫给予体型硫化剂，含有活性硫原子。

### 2. 核心性能特点 (Performance)

江苏锐巴的 MTT-80 主要作为传统有毒助剂（如 ETU/NA-22）的**环保替代品**，具有以下显著优势：

- **环保合规 (ETU-Free)**：
    - 这是 MTT-80 最大的卖点。它不含乙烯基硫脲（ETU），而 ETU 被欧盟 REACH 法规列为生殖毒性物质。使用 MTT-80 可以帮助橡胶制品企业满足严格的出口环保法规要求。
- **优异的交联效率**：
    - 对含卤素的聚合物具有极高的交联活性。
    - **硫化速度快**：能在较短时间内达到正硫化，提高生产效率。
- **加工性能优良**：
    - **无粉尘**：江苏锐巴采用造粒工艺，解决了粉末状助剂飞扬的问题，改善车间环境。
    - **易分散**：预分散母胶形式使其在橡胶基体中分散更均匀，避免局部过热或分散不良。
    - **不喷霜**：在常规用量下，不易发生喷霜现象。
- **物理机械性能好**：
    - 硫化后的胶料具有良好的拉伸强度、耐热性和压缩永久变形性能。

### 3. 主要应用领域 (Applications)

MTT-80 专用于**含卤素聚合物**的硫化交联，主要应用领域包括：

- **氯丁橡胶 (CR)**：
    - 这是最主要的应用场景。用于制造 CR 胶管、胶带、电缆护套、密封件、防震垫等。
    - 特别适用于对环保有要求的汽车配件和建筑材料。
- **氯化聚乙烯 (CPE)**：
    - 作为 CPE 橡胶的主硫化剂，提供优异的热稳定性和耐老化性。
- **氯醚橡胶 (CO / ECO)**：
    - 用于燃油胶管、隔膜等耐油耐臭氧制品。
- **氯化/溴化丁基橡胶 (CIIR / BIIR)**：
    - 用于轮胎内胎、硫化胶囊、药用瓶塞等气密性要求高的制品。

### 4. 推荐用量 (Dosage)

由于 MTT-80 是预分散母胶（80%含量），计算用量时需考虑载体重量。以下是基于纯品换算后的参考用量：

表格

|橡胶类型|推荐用量 (phr)|配合建议|
|---|---|---|
|**氯丁橡胶 (CR)**|**0.5 - 2.0 phr**|必须配合 **氧化镁 (MgO)** (防焦烧) 和 **氧化锌 (ZnO)** (活化剂) 使用。|
|**CPE / 氯醚橡胶**|**1.0 - 2.5 phr**|根据硬度要求调整，通常需配合金属氧化物。|

**💡 工艺建议**：

1. **并用体系**：在 CR 中，MTT-80 通常与金属氧化物（MgO/ZnO）并用。MgO 用于防止焦烧（早期硫化），ZnO 用于提供交联点。
2. **投料时机**：建议在密炼机混炼的中后期加入，或者在开炼机上加入，以确保分散均匀。
3. **替代 NA-22**：如果原配方使用 NA-22 (ETU)，通常 MTT-80 的用量需要略高于 NA-22（约 1.2:1 的比例）以达到同等的模量，具体需通过实验调整。

### 总结

江苏锐巴的 **MTT-80** 是一款**环保型、高效、易分散**的特种硫化剂，主要用于替代有毒的 ETU (NA-22)。如果你正在开发出口欧盟的**氯丁橡胶 (CR)** 或 **CPE** 制品，这款产品是确保合规性的理想选择。

**MTT-80** 与 **DETU** 虽然都属于橡胶助剂中的**硫脲类衍生物**，且都常用于特种橡胶，但它们在**化学结构**、**核心功能**以及**环保属性**上有本质的区别。

简单来说：**DETU 是传统的氯丁橡胶促进剂，但有毒（生殖毒性）；MTT-80 是环保型的交联剂，主要用于替代 DETU 或 ETU。**

### 2. 详细差异解析

#### **A. 功能定位不同**

- **DETU**：
    - 它主要作为**氯丁橡胶 (CR)** 的**促进剂**。
    - 它的特点是硫化起步快，硫化平坦性好，能提高胶料的抗张强度和硬度。
    - 它通常用于需要**高温高速硫化**的 CR 制品（如压出制品）。
- **MTT-80**：
    - 它主要作为**交联剂**（或称主硫化剂）。
    - 它不仅用于 CR，还广泛用于**氯化聚乙烯 (CPE)**、**氯醚橡胶 (ECO)** 和 **卤化丁基橡胶**。
    - 它利用分子中的活性硫原子直接参与交联反应，常与金属氧化物（氧化锌、氧化镁）配合使用。

#### **B. 环保与法规（最关键的区别）**

- **DETU**：
    - 属于硫脲类化合物，与 ETU (NA-22) 结构类似。这类物质在欧盟 REACH 法规中受到严格限制，被标记为**生殖毒性物质**。出口型橡胶制品（特别是汽车配件、食品接触类）正逐渐淘汰 DETU。
- **MTT-80**：
    - 被开发出来作为 **ETU (NA-22)** 和 **DETU** 的**环保替代品**。它不含乙烯基硫脲结构，毒性低，符合现代环保法规要求。

#### **C. 物理形态与加工**

- **DETU**：通常为白色粉末或晶体，容易吸湿，且在胶料中用量大时容易**喷霜**（Bloom）。
- **MTT-80**：市场上常见的 "MTT-80" 通常是指**预分散母胶**（80% 有效成分 + 20% 载体）。这种颗粒状形态**无粉尘**、**易分散**、**不喷霜**，更适合现代清洁生产工艺。

### 3. 应用建议：如何选择？

- **选择 DETU 的情况**：
    
    - 你的产品仅用于**国内低端市场**或非环保敏感领域。
    - 你需要极快的硫化起步速度，且配方体系是传统的 CR + 硫磺体系。
    - 成本控制非常严格（DETU 通常比新型环保助剂便宜）。
- **选择 MTT-80 的情况**：
    
    - 你的产品需要**出口**（特别是欧美市场），必须符合 REACH 法规。
    - 你正在生产 **CPE** 或 **氯醚橡胶** 制品（DETU 对这些胶种效果不佳）。
    - 你希望改善车间环境（无粉尘）并解决喷霜问题。
    - 你需要替代配方中的 NA-22 (ETU)。

### 总结

**MTT-80 是 DETU 的“进化版”和“环保版”。** 如果你在做氯丁橡胶 (CR) 或 CPE 配方，且面临环保压力，**强烈建议使用 MTT-80 替代 DETU**。

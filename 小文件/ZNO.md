**新际金属有限公司**（New Edge Metal Co., Ltd.，原美锌常熟金属有限公司）是一家位于江苏常熟的外商独资企业，专注于再生锌资源的循环利用，其主要产品为**间接法氧化锌**和再生锌锭。该公司的氧化锌产品以环保、高品质著称，广泛应用于橡胶、轮胎、陶瓷、化工及电子等行业。

### 1. 产品性能 (Performance)

新际金属生产的氧化锌主要采用**间接法**（法国法）工艺，利用再生锌原料提炼而成，具有以下显著性能特点：

- **高纯度**：主含量（ZnO）通常可达 **99.7% - 99.9%**，杂质含量极低。
- **低重金属含量**：由于采用先进的除尘和提纯技术，其铅（Pb）、镉（Cd）等有害重金属含量远低于国家标准，特别适合对环保要求严格的出口型橡胶制品、食品接触材料及高端化妆品应用。
- **物理形态**：白色细微粉末，无结块，粒径分布均匀，比表面积大，活性高。
- **化学稳定性**：常温下稳定，不溶于水和乙醇，溶于酸和碱。受热时颜色微黄，冷却后恢复白色。
- **分散性**：在橡胶和塑料基体中具有优异的分散性能，有助于提高最终产品的均一性。
- **环保优势**：作为再生锌产品，其生产过程符合循环经济理念，碳足迹相对较低，符合全球可持续发展的趋势。

**典型技术指标参考**（具体以厂家COA为准）

- **氧化锌** (ZnO) ≥ 99.7%
- **铅** (Pb) ≤ 0.05% (甚至更低，如≤0.02%)
- **镉** (Cd) ≤ 0.01%
- **筛余物** (45μm) ≤ 0.05%
- **加热减量**: ≤ 0.3%

### 2. 主要应用领域 (Applications)

新际金属的氧化锌凭借其低重金属和高活性的特点，主要应用于以下领域：

- **橡胶与轮胎工业**（核心应用）：
    - **硫化活性剂**：与硬脂酸反应生成硬脂酸锌，促进橡胶硫化，提高交联密度。
    - **补强剂**：提高橡胶制品的拉伸强度、撕裂强度和耐磨性。
    - **导热剂**：帮助轮胎在行驶过程中散热，延长使用寿命。
    - **适用对象**：全钢子午线轮胎、半钢轮胎、胶管、胶带、鞋底等。由于其低铅镉特性，特别适用于出口欧盟等对环保有严格限制的轮胎产品。
- **陶瓷与玻璃工业**：
    - **助熔剂**：降低釉料熔点，扩大熔融范围。
    - **乳浊剂与光泽剂**：增加釉面光泽度，改善质感。
    - **结晶剂**：用于生产结晶釉。
- **涂料与油漆**：
    - **防腐防锈**：作为防锈颜料，保护金属基材。
    - **防霉抗菌**：抑制霉菌生长，常用于船舶漆和外墙涂料。
    - **紫外线屏蔽**：吸收紫外线，防止涂层老化。
- **饲料与农业**：
    - **锌源补充**：作为动物饲料的营养强化剂（需符合饲料级标准），促进动物生长发育，增强免疫力。
    - **脱硫剂**：用于化肥工业原料气的精脱硫。
- **其他领域**：
    - **医药与化妆品**：用于软膏基质（如氧化锌软膏）、防晒霜（物理防晒剂）、爽身粉等（需使用高纯度、低重金属的特定型号）。
    - **电子材料**：用于压敏电阻、变阻器等电子元件的制造。
    - **塑料加工**：作为热稳定剂和润滑剂助剂。

### 3. 参考用量 (Dosage)

氧化锌的用量取决于具体的配方设计和产品性能要求，以下是各行业的常规参考用量：

- **橡胶制品**（轮胎/胶管/胶带）：
    - **常规用量**：**3.0 - 5.0 phr** (每百份橡胶中的份数)。
    - **低锌配方**：随着技术进步，部分高性能配方通过优化分散性或纳米化，可将用量降至 **1.5 - 2.5 phr**，同时保持性能并降低成本。
    - **配合使用**：通常与硬脂酸（1.0-2.0 phr）并用，两者反应生成可溶性锌盐以激活硫化体系。
- **陶瓷釉料**：
    - **用量范围**：**1.0% - 15.0%** (占釉料总重)。
    - **具体场景**：作为助熔剂时用量较少（约1-5%）；作为乳浊剂或结晶剂时用量较高（可达10-15%）。
- **涂料/油漆**：
    - **防锈漆**：用量通常为 **5.0% - 15.0%**。
    - **普通乳胶漆**：作为功能性填料，用量约为 **1.0% - 3.0%**。
- **饲料添加剂**：
    - **用量**：根据动物种类和生长阶段不同，通常在 **50 - 3000 g/吨** 饲料之间（即 0.005% - 0.3%）。
    - _注意：高剂量氧化锌（如2000-3000 ppm）曾用于仔猪腹泻防治，但受环保和耐药性法规限制，许多地区已限制高剂量使用，需遵循当地法规。_
- **医药/化妆品**：
    - **软膏/洗剂**：常用浓度为 **10% - 40%**。
    - **防晒霜**：纳米氧化锌用量通常为 **2.0% - 10.0%**。

### 4. 包装与储存

- **包装**：通常采用 **25kg/袋** 或 **500kg/袋**（吨袋）的编织袋包装，内衬塑料袋以防潮。
- **储存**：应存放于干燥、通风良好的仓库中，避免雨淋和受潮，远离酸类物质。
- **保质期**：在妥善储存条件下，保质期通常为 **24个月**。

**模得丽 935P** 是青岛昂记橡塑科技有限公司生产的一款高性能**内脱模剂**（Internal Mold Release Agent），同时也兼具分散剂和润滑剂的功能。它在橡胶加工行业中应用广泛，特别是在解决模具污染、提高脱模效率和改善胶料流动性方面表现优异。

以下是基于公开资料整理的详细性能、应用及用量指南：

### 1. 产品基本信息

- **产品名称**：模得丽 935P (Modelle 935P)
- **生产厂家**：青岛昂记橡塑科技有限公司
- **化学组成**：合成界面活性剂的金属皂基混合物（特殊复合配方）。
- **外观**：白色无尘粉粒（或白色粉末）。
- **物理性质**：
    - pH值：6.5 - 9.5
    - 加热减量：≤ 2.5%
    - 无机含量：≤ 36%
    - 包装规格：通常为 25kg/袋（牛皮纸袋内衬塑料袋）。
- **储存条件**：防潮、阴凉干燥处保存。

### 2. 核心性能与功能

模得丽 935P 不仅仅是一个脱模剂，它在混炼和硫化过程中发挥多重作用：

1. **卓越的脱模性**：
    
    - 在硫化过程中迁移到胶料表面，形成极薄的隔离层，显著降低胶料与模具表面的粘附力。
    - **效果**：使制品易于脱模，减少因强行脱模造成的制品变形或撕裂，特别适用于复杂结构和精密模具。
2. **减少模具污染（防积垢）**：
    
    - 能中和胶料中可能产生沉积物的酸性成分或副产物。
    - **效果**：大幅减少模具表面的积垢（模垢），延长模具清洗周期，减少停机清洗时间，提高生产效率。制品表面光亮，无喷霜现象。
3. **改善加工性能（润滑与分散）**：
    
    - 在混炼阶段起到内润滑作用，降低胶料粘度，改善填料（如炭黑、白炭黑）的分散性。
    - **效果**：降低能耗，防止焦烧，提高胶料的流动性和填充性，减少缺胶现象。
4. **不影响物理性能**：
    
    - 与传统外涂脱模剂不同，作为内添加型助剂，适量使用不会影响橡胶制品的拉伸强度、撕裂强度、老化性能等关键物理指标。
    - 在氟橡胶等特种橡胶中的应用研究表明，它不会降低胶料的物性。
5. **环保与安全**：
    
    - 不含硅酮（Silicone-free），避免了对后续涂装、粘接工艺的干扰（硅酮会导致油漆剥落或粘接失败）。
    - 无毒、无味，符合环保要求。

### 3. 主要应用领域

模得丽 935P 适用于多种橡胶体系，尤其在以下领域表现突出：

- **氟橡胶 (FKM)**：这是其最经典的应用场景。氟橡胶加工难度大、易粘模，935P 能显著改善其流动性和脱模性，且不影响其耐高温和耐化学腐蚀性能。
- **硅橡胶 (VMQ)**：用于改善挤出和模压工艺，减少模具沉积。
- **丁腈橡胶 (NBR) / 氢化丁腈 (HNBR)**：广泛用于油封、O型圈、胶管等制品的生产。
- **三元乙丙橡胶 (EPDM)**：用于汽车密封条、门窗胶条等，提高表面光洁度。
- **天然橡胶 (NR) / 丁苯橡胶 (SBR)**：用于轮胎部件、减震制品等。
- **其他**：氯丁橡胶 (CR)、丙烯酸酯橡胶 (ACM) 等。

**典型制品**：

- O型圈、油封、垫片等密封件。
- 汽车胶管、传动带。
- 电线电缆护套。
- 鞋底、滚轮等模压制品。

### 4. 推荐用量与使用方法

#### **推荐用量**

- **常规用量**：**1.5 - 4.0 phr** (份/百份橡胶)。
    - 大多数通用橡胶（如 NBR, EPDM, SBR）：**2.0 - 3.0 phr**。
    - 氟橡胶 (FKM) 或高难度脱模配方：**3.0 - 4.0 phr**。
    - 轻微脱模需求或作为分散剂辅助：**1.5 - 2.0 phr**。
- **注意**：过量使用可能导致制品表面过于光滑而影响二次加工（如粘接、印刷），或导致成本增加。建议通过小试确定最佳用量。

#### **使用方法**

1. **加入时机**：
    - 通常在**混炼初期**或**中期**加入。
    - 建议在生胶塑炼后，加入填充剂（炭黑/白炭黑）之前或同时加入，以便更好地分散。
    - 也可以在终炼阶段加入，但需确保分散均匀。
2. **混合工艺**：
    - 直接以粉末形式加入密炼机或开炼机中。
    - 由于其为白色无尘粉粒，分散性较好，一般不需要预先母胶化处理。
3. **兼容性**：
    - 与大多数橡胶配合剂（硫化剂、促进剂、防老剂等）相容性好，无明显副作用。
    - **注意**：如果制品需要进行后续的**粘接处理**（如与金属骨架粘接），建议先进行小试验证，确认不影响粘合性能（虽然它不含硅，但任何脱模剂都可能对粘接有微弱影响）。

### 6. 注意事项

1. **储存**：务必保持干燥，受潮后可能结块，影响分散。
2. **用量控制**：并非越多越好，过量可能导致制品表面析出（虽然比硬脂酸锌好很多），影响外观和粘接。
3. **特殊工艺**：对于需要高温二次硫化（如氟橡胶、硅橡胶）的制品，935P 具有良好的热稳定性，不会在高温下分解失效。
4. **采购渠道**：请通过青岛昂记官方或其授权经销商购买，以确保正品和获得技术支持。市场上存在仿冒品，性能差异较大。

### 总结

**模得丽 935P** 是一款高效、多功能的**内脱模剂**，特别适合对模具清洁度、制品表面质量和生产效率有较高要求的橡胶加工企业。

- **核心价值**：省去了外喷工序，减少了洗模次数，提高了良品率。
- **最佳拍档**：氟橡胶、硅橡胶、以及各类高精度密封件的生产。
- **建议用量**：从 **2-3 份** 开始尝试，根据具体配方和脱模效果微调。

如果您正在面临**粘模、模具脏得快、制品表面不光亮**等问题，模得丽 935P 是一个非常值得尝试的解决方案。建议先索取样品进行小批量试验，以确定在您特定配方中的最佳用量。

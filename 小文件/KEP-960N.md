**KEP-960N（Kumho KEP-960N）** 是韩国 **Kumho Polychem（锦湖石化）** 生产的**充油型（Oil extended）** EPDM生胶，属于**高乙烯 + 中等ENB + 中等分子量分布**的牌号，使用非芳香族石蜡油（non-aromatic paraffinic oil）充油。

### 主要性能参数（典型值，基于PDS）

- **门尼粘度**：ML(1+8) 125°C = **49.0 ± 6.0 MU**（充油后，较高门尼）
- **乙烯含量 (Ethylene)**：**70.0 ± 3.0 wt%**（高乙烯）
- **ENB（第三单体，二烯）含量**：**5.7 ± 1.0 wt%**（中等ENB）
- **充油量**：**50.0 ± 3.0 phr**（约33-35%油含量，具体视批次）
- **挥发分**：≤ **0.75 wt%**（ASTM D5668）
- **分子量分布**：**Medium（中等）**
- **密度**：约 **0.86-0.88 g/cm³**（充油后）
- **物理形态**：胶块（Bale），通常25 kg/块或35 kg/块，用PE膜或EOM膜包裹，纸箱/网箱包装（约600-840 kg/托）
- **其他**：添加非污染型抗氧剂，RoHS符合

**关键特性**：

- **高乙烯含量** → 结晶倾向适中，物理机械性能优异（拉伸强度、模量、硬度保持好）
- **高填充能力（High loadability / high filler loading）** → 可加入大量炭黑/填料仍保持良好性能
- **良好加工性（Superior processability）** 和挤出性能
- **与SBR共混时物理性能好**
- **优异UV/耐候稳定性**（EPDM通用优势）
- **压缩永久变形控制较好**，耐热老化、耐臭氧、耐水性突出
- 充油后加工流动性改善，成本可控，同时基础聚合物高分子量使成品强度较高

高乙烯 + 充油50 phr的设计，使其适合需要**较高物理性能 + 挤出稳定性 + 成本平衡**的场合。

### 主要应用领域

KEP-960N 主要推荐用于挤出制品和模压件，特别是要求**尺寸稳定、物理性能良好**的场景：

- **汽车工业**：窗密封条（window seals）、天气密封条、汽车挤出型材、汽车零部件
- **家电**：洗衣机门密封圈（washing machine door gasket）、电器密封件
- **建筑/门窗**：门窗密封条、垫圈（gaskets）、密封制品
- **其他**：一般挤出制品（extruded goods）、模压件（moldings）、电器/家电橡胶件

特别适合**窗密封条、汽车密封**等挤出制品，以及与SBR等并用以提升整体性能的配方。

### 用量建议（配方参考）

在典型EPDM配方中，**KEP-960N 常作为主胶或唯一生胶使用，用量为 100 phr**。

#### 典型挤出密封条/窗密封配方示例（仅供参考，实际需小试调整）：

- KEP-960N：**100 phr**
- ZnO（氧化锌）：4-5 phr
- 硬脂酸：1-2 phr
- PEG 4000（分散剂）：2-4 phr
- 炭黑（N550/N660 或 FEF，根据硬度）：80-150 phr（可高填充，利用其高负载能力）
- 碳酸钙/白填料：50-120 phr
- 额外石蜡油/操作油：20-60 phr（视目标硬度调整，已充油50 phr为基础）
- 防老剂（TMQ、6PPD 或 UV稳定剂）：1-2 phr
- 硫化体系：硫磺 1.0-1.8 phr + 促进剂（MBTS + TMTD 或 TBBS 等） + 助硫化剂
- 其他：CaO（吸湿剂）等

**优点体现**：

- 高乙烯提供较好的挤出尺寸稳定性和表面质量。
- 高填充时仍能保持良好机械性能（如拉伸强度、硬度）。
- 成品硬度可控制在Shore A 50-70左右，适合汽车/家电密封件。
- 与SBR并用（例如20-50 phr）时，整体物理性能和成本平衡佳。

**注意事项**：

- 储存：置于阴凉干燥处（<30°C），避免阳光直射和潮湿。原包装保存，保质期约24个月。
- 加工：混炼时利用充油优势，控制温度以避免过早硫化；挤出时尺寸稳定性好。
- 实际批次数据可能有小幅波动（门尼、乙烯/ENB在公差范围内），**强烈建议向Kumho Polychem或当地代理索取最新PDS/TDS和安全数据表（SDS）**，并进行实验室验证（重点测试门尼、硫化曲线、压缩永久、挤出塌陷、耐老化等）。
- 与之前讨论的牌号简要对比：
    - **vs KEP-330**：KEP-330是非充油、低乙烯（~57%）、高ENB（~7.9%）、低门尼（~28）→ 加工流动性和硫化速度更快，适合海绵/快速生产；KEP-960N是充油、高乙烯（70%）、中等ENB、中高门尼 → 物理性能和挤出稳定性更好，适合窗密封等要求强度的制品。
    - **vs 4045M (Mitsui)**：4045M是非充油、低乙烯（~45%）、高ENB、窄分布 → 低温柔韧性更优；KEP-960N高乙烯 + 充油，更注重高温性能和高填充。
    - **vs TER4436 (Versalis)**：两者都是充油型，TER4436充油~40%、乙烯聚合物基较高分子量；KEP-960N充油50 phr、高乙烯70%，在物理性能和家电/汽车密封上有类似适用性。

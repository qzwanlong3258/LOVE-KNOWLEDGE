**KEP-330（Kumho KEP-330）** 是韩国 **Kumho Polychem（锦湖石化）** 生产的**非充油（Non-oil extended）** EPDM生胶，属于**低乙烯 + 高ENB + 低门尼粘度**的牌号，分子量分布为Medium（中等）。它是EPDM系列中加工性和硫化速度突出的通用/挤出级产品。

### 主要性能参数（典型值）

- **门尼粘度**：ML(1+4) 125°C = **28 ± 5 MU**（低门尼粘度，部分资料标ML1+4@100°C类似范围；聚合物在140°C重磨后测试）
- **乙烯含量 (Ethylene)**：约 **57%**（56.5 ± 3.5 wt%，属于低乙烯范畴）
- **ENB（第三单体，二烯）含量**：约 **7.9 ± 1.0 wt%**（高ENB）
- **分子量分布**：**Medium（中等）**
- **充油量**：**0 phr**（纯胶，非充油）
- **密度 / 比重**：约 **0.86 g/cm³**
- **挥发分**：≤ 0.75 wt%（ASTM D5668）
- **灰分**：低（典型EPDM水平）
- **物理形态**：胶块（Bale），通常25 kg/块，用PE膜包裹
- **其他**：添加非污染型抗氧剂，适合与二烯类橡胶共混

**关键特性**：

- **低门尼粘度 + 优异加工性**（excellent processability，流动好、易混炼、易挤出）
- **超快硫化速度**（ultra fast vulcanization speed，高ENB带来高交联速率）
- **低温性能良好**（低乙烯设计使柔韧性佳）
- **适合与二烯橡胶（如NR、SBR、BR）共混**，改善耐臭氧性能
- EPDM通用优势：优异耐候、耐臭氧、耐热老化、耐水、耐化学品、电绝缘性好、低压缩永久变形潜力
- 与高乙烯牌号相比，更柔软、加工流动更好；与低ENB牌号相比，硫化更快、生产效率更高

低门尼 + 高ENB + 中等分子量分布的组合，使其在挤出和海绵制品中表现突出，加工窗口宽，成品表面质量好。

### 主要应用领域

KEP-330 主要推荐用于需要**快速硫化、良好加工性**的挤出和模压制品：

- **轮胎工业**：胎侧白侧壁（white sidewalls）、盖条（cover strips）
- **汽车工业**：窗密封条（window seals）、天气密封条（weatherstrip solid）、汽车零部件、挤出型材
- **建筑/门窗**：密封条、垫圈（gaskets）、填料（packing）
- **其他**：海绵制品（sponge）、耐热软管、电缆护套、工业橡胶件、挤出制品

特别适合**海绵、密封条、与二烯橡胶共混**的场合，能提高生产速度和产品一致性。

### 用量建议（配方参考）

在典型EPDM配方中，**KEP-330 常作为主胶或唯一生胶使用，用量为 100 phr**（可与少量其他橡胶并用）。

#### 典型挤出/密封条或海绵配方示例（仅供参考，实际需小试调整）：

- KEP-330：**100 phr**
- ZnO（氧化锌）：4-5 phr
- 硬脂酸：1-2 phr
- PEG 4000（分散剂）：2-4 phr
- 炭黑（N550/N660 或轻质填料，根据硬度）：60-120 phr（海绵可降低）
- 碳酸钙/白填料：50-150 phr
- 石蜡油/操作油：30-80 phr（调节硬度和加工性）
- 发泡剂（海绵制品，如OBSH或ADC）：适量 + 助发泡剂
- 防老剂（TMQ、6PPD 等）：1-2 phr
- 硫化体系：硫磺 1.0-1.8 phr + 促进剂（MBTS + TMTD 或 CBS/TBBS 等，高ENB下促进剂用量可适当降低） + 助硫化剂
- 其他：CaO（吸湿剂）、发泡稳定剂等

**优点体现**：

- 超快硫化速度显著提高生产效率，减少焦烧风险。
- 低门尼使挤出速度快、表面光洁、尺寸稳定。
- 海绵制品发泡均匀，密封条压缩永久变形控制好。
- 与二烯橡胶共混时（如10-30 phr NR/SBR），可提升整体耐候性。

**注意事项**：

- 储存：置于阴凉干燥处（<30°C），避免阳光直射和潮湿。原包装保存，防止结晶或老化。
- 加工：混炼温度控制得当，利用其低粘度优势；硫化温度可略低以匹配超快速度。
- 实际批次数据可能有小幅波动（乙烯/ENB在公差范围内），**强烈建议向Kumho Polychem或当地代理索取最新TDS/PDS和安全数据表（SDS）**，并进行实验室小试（重点验证门尼、硫化曲线（t90很短）、压缩永久、低温性能、挤出塌陷等）。
- 与之前讨论的牌号简要对比：
    - **vs 4045M (Mitsui)**：两者都是低乙烯 + 高ENB + 非充油，4045M窄分布、门尼稍高（~45），低温性能类似；KEP-330门尼更低（28）、硫化更快、加工性更优。
    - **vs TER4436 (Versalis)**：TER4436是充油40%、较高乙烯、高分子量 → 更适合高填充、低硬度；KEP-330非充油、加工流动和硫化速度更突出，适合纯胶或中填充挤出。
    - **vs KEP-960N**：KEP-960N是高乙烯 + 充油，物理性能和挤出稳定性更好；KEP-330更注重加工速度和共混性。

### 详细性能与用途差异分析

1. **4045M (Mitsui EPT 4045M)**
    - 低乙烯 + 高ENB + 窄分子量分布 → **低温性能出色**（柔韧性好、耐寒），加工流动性佳。
    - 适合需要良好挤出性、注射成型和低温弹性的产品，如汽车密封条、海绵等。
    - 非充油，纯胶基，硫化后交联密度较高，耐热老化与加工平衡好。
2. **TER4436 (Dutral TER 4436)**
    - **充油型**（40%油），高分子量、半结晶结构。
    - 优点：高填充能力（可加大量填料仍保持性能）、绿色强度好、适合低硬度制品、挤出尺寸稳定性强、硫化速率适中。
    - 适合汽车零部件、机械橡胶件、TPV改性等需要高填充和低成本的场景。充油后门尼粘度虽低，但基础聚合物分子量高。
3. **KEP-330 (Kumho KEP330)**
    - 低门尼粘度 + 高ENB → **加工性极佳**、**硫化速度非常快**。
    - 低乙烯设计使低温性能和柔软性较好，适合与天然胶/丁苯胶等二烯类橡胶共混。
    - 典型用于窗密封、垫圈、海绵、白侧壁等要求快速硫化和良好流动性的制品。
4. **KEP-960N (Kumho KEP960N)**
    - **高乙烯 + 充油** → 物理机械性能较好（拉伸强度、硬度保持好）、耐热性较优。
    - 高乙烯使结晶度稍高，适合挤出制品（如窗密封条、汽车密封件）。
    - 充油后加工流动性改善，成本控制较好，适用于电器、汽车挤出件。

### 选择建议（根据需求对比）

- **追求低温柔韧性 + 加工性**：优先 **4045M** 或 **KEP-330**（两者低温与流动好）。
- **需要高填充 + 低硬度 + 成本控制**：选 **TER4436**（充油高分子量优势明显）。
- **高物理性能 + 挤出稳定性**：选 **KEP-960N**（高乙烯特点）。
- **硫化速度要求快**：**KEP-330** 最突出（超快硫化）。
- **非充油纯胶**：**4045M** 和 **KEP-330** 更合适；充油型（TER4436、KEP-960N）更易加工和高填充。

**注意事项**：

- EPDM通用优点：耐候、耐臭氧、耐热（-40~150°C左右）、电绝缘好。
- 实际配方中，充油型可降低成本但可能略影响某些动态性能；非充油型纯度更高。
- 建议根据具体应用（如汽车密封 vs 海绵 vs 电缆）咨询供应商索取最新TDS，并做小试验证硫化性能、压缩永久变形等。

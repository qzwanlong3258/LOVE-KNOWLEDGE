**Peroxide 橡胶认证**（过氧化物硫化橡胶认证）是指使用**过氧化物（如DCP、双叔丁基过氧化物、BIPB等）**作为交联剂/硫化剂的橡胶制品，需要通过的相关安全、环保和应用认证。

过氧化物硫化（Peroxide Cure）与传统硫磺硫化（Sulfur Cure）相比，交联键为**C-C键**（碳-碳键），具有**耐热性更好、压缩永久变形更低、耐化学品性更强、无硫残留（不污染金属、不产生硫化返原）**等优点，但成本较高，且分解产物可能有轻微气味或残留物。

### 1. 过氧化物硫化橡胶的主要优势（为什么需要认证）

- 耐高温（通常可达150–200°C以上）
- 低压缩永久变形（适合密封件）
- 优异的耐老化、耐蒸汽、耐热水性能
- 无硫残留（不腐蚀铜、银等金属，不污染食品/药品接触面）
- 常用于EPDM、硅橡胶（VMQ）、HNBR、某些NBR、EVM（Levapren）等

缺点：动态疲劳性能有时略逊于硫磺硫化；部分过氧化物分解后可能产生苯乙酮等副产物，需充分**后硫（Post Cure）**以降低残留。

### 2. 常见认证类型及要求

过氧化物硫化橡胶的认证主要取决于应用领域：

|应用领域|主要认证标准|具体要求与注意事项|常见适用橡胶类型|
|---|---|---|---|
|**食品接触**|**FDA 21 CFR 177.2600**（美国） EC 1935/2004（欧盟）|提取物测试（蒸馏水、己烷等溶剂）；需充分后硫降低挥发物；很多过氧化物硫化EPDM/硅胶可通过|EPDM、硅橡胶、EVM|
|**饮用水接触**|**NSF/ANSI 61**、**WRAS**（英国）|饮用水系统组件认证；对迁移物要求严格|Peroxide-cured EPDM|
|**食品级欧洲**|**LFGB**（德国）、**BfR**|感官测试 + 迁移测试；过氧化物体系需严格控制残留|硅橡胶、EPDM|
|**医用/制药**|**USP Class VI**、**ISO 10993**|生物相容性测试（细胞毒性、敏感性等）；铂金硫化（Platinum）更容易通过，过氧化物体系需强后硫|硅橡胶（部分过氧化物可通过）|
|**环保/有害物质**|**RoHS**、**REACH**（SVHC）|限制铅、汞、镉、六价铬、特定阻燃剂等；DCP本身可能被关注，但成品通常可符合|所有橡胶制品|
|**阻燃**|**UL94 V-0 / HB**|垂直/水平燃烧测试|电缆用EPDM、硅橡胶|
|**其他工业**|**ASTM D2000**、**ISO 9001**|性能分级（如耐热、耐压缩永久）；质量管理体系|通用密封件、O型圈|

**重要提示**：

- **过氧化物硫化 vs 铂金硫化（Platinum Cure）**： 铂金硫化（加成型）残留物更少、纯度更高、更容易通过**FDA、USP Class VI、LFGB**等高端认证，常用于医用和高端食品接触。 过氧化物硫化成本更低、耐热和压缩永久往往更好，但需**充分后硫（Post Cure，通常200°C×几小时）**来降低挥发物和残留，才能满足严格的食品/医用认证。
- **DCP（过氧化二异丙苯）体系**：许多DCP硫化的EPDM可通过FDA 177.2600（橡胶制品反复接触食品）。但需注意分解产物（如苯乙酮），建议进行迁移测试。

### 3. 实际建议（橡胶工厂视角）

- **食品/饮用水接触**：优先选择**过氧化物硫化EPDM**或**硅橡胶**，并要求供应商提供**FDA声明 + 第三方测试报告**。
- **医用级**：强烈推荐**铂金硫化**硅橡胶，更容易通过USP Class VI。
- **普通工业级**（密封条、电缆、汽车件）：RoHS + REACH + ASTM D2000 即可，过氧化物体系优势明显。
- **认证流程**：配方固定后 → 送第三方实验室测试提取物/迁移物 → 获得符合性声明（Compliance Statement）或证书。
